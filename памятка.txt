cd (change directory) - сменить папку (.. - назад)

git add . - добавление всех файлов в отслеживание
git commit -m "описание комита" - сохранение (feat, fix)
git push - отправление в репозиторий

git checkout название_ветки - переход по веткам (git checkout -b название_ветки - в новую ветку)

git pull - достатьс последние изменения с сервера
